# 자릿수 더하기

## 문제 설명
자연수 `N`이 주어지면, `N`의 각 자릿수의 합을 구해서 return 하는 `solution` 함수를 만들어 주세요.
예를들어 `N` = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

## 제한 사항
- `N`의 범위 : 100,000,000 이하의 자연수

## 코드
```java
import java.util.*;

public class Solution {
    public int solution(int n) {
        int answer = 0;

        while (n != 0) {
            int a = n % 10;
            answer += a;
            n /= 10;
        }

        return answer;
    }
}
```
